# Disaster Management System

This project is a Disaster Management System developed using Spring Boot.

## 📌 Description
The application helps manage disaster-related data, reports, and administrative operations efficiently.

## 🚀 Features
- Admin dashboard
- Manage disaster data
- Generate reports
- Secure configuration

## 🛠️ Technologies Used
- Java
- Spring Boot
- Maven
- HTML, CSS

## 📂 Project Structure
- src/main/java → Backend code
- src/main/resources → Configuration & static files
- pom.xml → Project dependencies

## ▶️ How to Run
1. Import project into Eclipse
2. Run as Spring Boot Application
3. Open browser: http://localhost:9595/login.html
